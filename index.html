1. Harnessing AI & Low-Code for Impact

Convert text ideas into small, deployable apps that address SDG 2 (Zero Hunger) in Nigeria. This static demo is fully deployable on GitHub Pages; sensitive calls (payments, real AI keys) should be handled by a secure backend.

2. Areas the Hackathon is focusing on

AI Study Buddy — Flashcards (Education spin)

Generate study flashcards from notes (frontend-only demo or call an API from backend).

  • Frontend: HTML5 / CSS / JS
  • Backend (opt): Flask (Render) / Node (Render)
  • AI: Hugging Face Q&A (server-side)

Farm-to-Table (Primary SDG2 focus)

Connect farmers with buyers to reduce waste and improve incomes.

  • Frontend: HTML / JS listings
  • Backend (opt): Flask / Node for orders & secure payments
  • Payments: IntaSend (backend integration)

Nutrition Coach (Low-cost meals)

Suggest affordable meal plans using local staples (cassava, rice, maize).

4. Monetization with IntaSend

IntaSend is recommended for African markets — it handles local payments and scales with growth. This demo simulates a checkout flow client-side; in production, implement IntaSend server-side.

Note: Do NOT place real keys in frontend code. Use a backend to create/checkout payments securely.
